How to use the application

Step 1

Create a Jira Scrum Project and link it to the KPI Dashboard.


The project must have Sprints, with tasks on it and must be public.


For a better use and a better aproximation of the KPIs tracked, is convenient to track the spent time on each task, perform a task estimation, report found bugs and everything related to the project.

Step 2

Once you have logged in, check if your project corresponds to the Jira Board project.


You can track some general aspects such as the members of your team, start and end dates, and a few more.


Then, navigate to Dashboards and you will see four available. Select the one you want and you will navigate to the selected Dashboard.


On the Dashboard you will see some metrics tracked and computed by our technology. You can export the dashboard by clicking on the Export button
